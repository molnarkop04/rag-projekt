# RAG Projekt: Intelligenter Dokumentenassistent

Es wird eine RAG-Applikation entwickelt, die es Nutzer ermöglicht, mit ihren eigenen PDF-Dokumenten in natürlicher Sprache zu kommunizieren. Ein Assistent wird entwickelt, der den inhaltlichen Kontext versteht. Er Funktioniert als ein intelligenter Analyst, der ein Dokument in Sekunden liest und spezifische Fragen dazu basierend auf Fakten beantwortet.

## Die Kernfunktionen

* Der Nutzer kann ein beliebiges PDF-Dokument über eine Web-Oberfläche hochladen
* Die App versteht die Bedeutungen einer Frage
* Die KI generiert Antworten ausschließlich auf Basis des hochgeladenen Dokuments
* Die App zeigt an, aus welchem Abschnitt des Dokuments  die Informationen stammen

## Das technische Ziel

### End-to-End Pipeline bauen, die

* PDF-Datei in maschinenlesbare Vektoren umwandelt
* Diese Vektoren effizient speichert und durchsuchbar macht
* Ein Large Language Modell als Sprach-Engine nutzt, um gefundene Informationen für den Nutzer verständlich und korrekt zusammenzufassen
